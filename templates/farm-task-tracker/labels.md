# Labels Configuration

> Use these labels to categorize tasks in your farm repository.
> To create these labels: go to Issues → Labels → New label.

| Label Name | Color (Hex) | Description |
|-----------|-------------|-------------|
| `equipment` | `#0075CA` | Tractors, irrigation, fencing, tools |
| `livestock` | `#2DA44E` | Cattle, sheep, poultry, horses |
| `crops` | `#E4E669` | Planting, harvest, pest, disease |
| `maintenance` | `#D876E3` | Routine upkeep, cleaning, repairs |
| `urgent` | `#E11D48` | Needs immediate attention |
| `routine` | `#BFD4F2` | Regular scheduled tasks |
| `compliance` | `#7057FF` | Grants, inspections, certifications |
| `seasonal` | `#B08800` | Planting season, harvest, winter prep |
| `safety` | `#FF6F61` | Safety concerns or hazards |
| `supplies` | `#1D76DB` | Supply orders and inventory |
