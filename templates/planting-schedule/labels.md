# Labels Configuration

> Use these labels to categorize planting tasks in your schedule repository.
> To create these labels: go to Issues → Labels → New label.

## Crop Type Labels

| Label Name | Color (Hex) | Description |
|-----------|-------------|-------------|
| `corn` | `#E4E669` | Corn plantings |
| `soybeans` | `#2DA44E` | Soybean plantings |
| `wheat` | `#B08800` | Wheat plantings |
| `vegetables` | `#E11D48` | Garden vegetables |
| `cover-crop` | `#1A7F37` | Cover crop plantings |
| `hay-pasture` | `#6E7781` | Hay and pasture management |
