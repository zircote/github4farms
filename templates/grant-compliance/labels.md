# Labels Configuration

> Use these labels to categorize grant compliance tasks.
> To create these labels: go to Issues → Labels → New label.

| Label Name | Color (Hex) | Description |
|-----------|-------------|-------------|
| `milestone` | `#2DA44E` | Major grant deliverables |
| `reporting` | `#0075CA` | Periodic reports and submissions |
| `documentation` | `#E4E669` | Supporting documents to gather |
| `financial` | `#7057FF` | Budget items, receipts, invoices |
| `site-visit` | `#E68D09` | Inspector or auditor visits |
| `overdue` | `#E11D48` | Past deadline |
| `upcoming` | `#BFD4F2` | Due within 30 days |
| `complete` | `#6E7781` | Deliverable met |
| `phase-1` | `#B08800` | First grant phase |
| `phase-2` | `#D4A373` | Second grant phase |
