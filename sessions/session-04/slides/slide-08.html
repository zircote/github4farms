Editing a File on GitHub

Edit any file directly on GitHub â€” no special software needed.

  1. Navigate to the file you want to change
  2. Click the pencil icon (top right) to edit
  3. Make your changes in the text editor
  4. Add a short commit message describing your change
  5. Click "Commit changes"

The commit message is like a sticky tag: "Updated planting dates for Field 2."
