Merging a Pull Request

When the review is done, the owner merges the changes.

On the Pull Request page: Click "Merge pull request" → "Confirm merge"

After merging:

  • The changes become part of the original repository
  • The Pull Request is marked "Merged" (purple)
  • Full history preserved — who, what, when

Like the farm manager saying: "Great suggestion — I've updated the official plan."
