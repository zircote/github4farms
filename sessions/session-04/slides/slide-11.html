Reviewing a Pull Request

The reviewer can:

  • Read the changes (green = added, red = removed)
  • Comment — ask questions or suggest edits
  • Approve — "Looks good, let's use these changes"
  • Request changes — "Please fix this part first"

To leave a review: Open the PR → Click "Files changed" → Click "Review changes" → Choose Comment / Approve / Request changes → "Submit review"
