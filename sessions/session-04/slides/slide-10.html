Creating a Pull Request

After editing your fork, create a Pull Request:

  1. Go to your forked repository
  2. Click "Contribute" → "Open pull request" (or use the Pull requests tab)
  3. GitHub shows what changed (green = added, red = removed)
  4. Add a title and description explaining the change
  5. Click "Create pull request"
