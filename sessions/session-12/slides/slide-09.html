Getting Started: Your First 5 Minutes

  1. Go to github.com and click + > New repository
  2. Name it something descriptive (e.g., sunrise-acres-crop-plan)
  3. Check "Add a README file"
  4. Click "Create repository"
  5. Edit the README to describe your farm in 2-3 sentences

You should be at this point within 5 minutes. Then move to creating Issues.
