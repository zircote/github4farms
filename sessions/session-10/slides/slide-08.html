How to Access Copilot in GitHub

Accessing Copilot on the Web:

  1. Go to your repository on GitHub
  2. Click the Issues tab
  3. Click "New issue"
  4. In the description box, look for the Copilot icon (sparkle icon)
  5. Click the icon or start typing for suggestions

You may also see Copilot in Pull Request descriptions, comment boxes, and the Copilot Chat sidebar.
