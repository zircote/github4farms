Reading a YAML File (Read-Only)

The workflow instructions are written in a file called YAML.

YAML is like a recipe card for your automatic helper. Today, you will learn to read a YAML file, not write one.

# Recipe name

name: Weekly Equipment Reminder

# When to start

on:

  schedule:

    cron: '0 8 * * MON' # Mon 8AM

# The work to do

jobs:

  send-reminder:

    steps:

      name: Create reminder Issue

Don't worry about every line. Focus on the parts we will highlight next.
