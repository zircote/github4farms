Guided Practice Steps

Follow along with me:

  1. Go to the demo repository (URL on your handout)
  2. Click the "Actions" tab
  3. Find a workflow run with a green checkmark and click it
  4. Answer: What triggered this workflow? When did it run?
  5. Find a run with a red X — what does that tell you?
  6. Navigate to .github/workflows/ in the code
  7. Click a .yml file — spot the name, trigger, job, and steps

Checkpoint: Raise your hand when you can point to the name, trigger, and steps in the YAML file.
