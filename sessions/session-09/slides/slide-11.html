Linking Issues from Multiple Repositories

One Project, Many Barns

You might have separate repos for crops, equipment, livestock, and business. A GitHub Project can pull Issues from all of them into one view.

How to add Issues from another repository:

  1. Open your Project
  2. Click "Add item" (the "+" at the bottom)
  3. Type # and the repository name (e.g., #equipment)
  4. Click the Issue to add it
