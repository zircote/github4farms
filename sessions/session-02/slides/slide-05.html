What Is an Issue?

Issue = Digital Work Order

A GitHub Issue is a task, job, or item that needs attention. Each Issue has:

  • Title — What needs to be done (short and clear)
  • Description — Details about the task
  • Labels — Color-coded tags to categorize it
  • Assignee — Who is responsible
  • Status — Open (needs doing) or Closed (done)
  • Comments — Updates and discussion
